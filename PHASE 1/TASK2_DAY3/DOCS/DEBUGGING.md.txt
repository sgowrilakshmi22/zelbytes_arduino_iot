# Serial Monitor Debugging Workflow

## Opening Serial Monitor

1. Connect Arduino using USB.
2. Upload the sketch.
3. Open Tools → Serial Monitor.
4. Select baud rate 9600.

## Button LED Output

Button LED Toggle Started

LED State: ON

LED State: OFF

## Relay Valve Output

Relay Valve Control Ready

Valve ON

Valve OFF

## Troubleshooting

### No Serial Output

* Check USB cable
* Verify COM/Port selection
* Verify board selection
* Confirm baud rate is 9600

### Button Not Working

* Check D2 connection
* Verify GND connection

### Relay Not Switching

* Check D7 connection
* Verify relay power connections
* Verify external power supply
