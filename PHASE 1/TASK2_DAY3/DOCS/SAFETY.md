# Safety Guidelines

## Solenoid Valve Safety

1. Never connect a solenoid valve directly to an Arduino GPIO pin.
2. Always use a relay module for isolation.
3. Use an external power supply for the valve.
4. Disconnect power before modifying wiring.
5. Verify all connections before turning power on.
