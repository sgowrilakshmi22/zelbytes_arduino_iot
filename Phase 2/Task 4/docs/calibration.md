
# Calibration Notes

## Soil Moisture

1. Measure sensor value in dry air.
2. Record as DRY value.

Example:

DRY = 850

3. Insert probe into fully wet soil.

Example:

WET = 300

Store these values inside sketch.

---

## LDR

Record values under:

- Bright sunlight
- Room lighting
- Darkness

Example:

Bright = 250

Normal = 550

Dark = 850

Use these values as thresholds.
